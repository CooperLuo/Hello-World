四个正在发生的趋势

① 算力军备竞赛

  • 万卡集群 → 十万卡集群
  • 电力 / 散热 成为新瓶颈
  • "主权 AI"驱动各国自建

② 模型趋于平价

  • 开源追平闭源差距
  • Token 价格 1 年降 90%+
  • "模型是商品，数据是壁垒"

③ Agent 元年

  • 从 Chat 到 Act：会用工具的 AI
  • 编程、客服、运营率先落地
  • Computer Use / Browser Use

④ 具身智能升温

  • 人形机器人量产元年
  • 世界模型 + 大模型 + 硬件
  • 制造、物流、家庭场景

瓶颈与风险

  • 能源：一座大型 AI 数据中心 ≈ 一座中型城市用电量
  • 先进制程：CoWoS 封装产能 90% 集中在台积电
  • 数据枯竭：高质量公开语料即将"训练完"，依赖合成数据
  • 商业化：模型公司单位经济模型 (Unit Economics) 仍待证明
  • 监管：版权、隐私、安全、出口管制（A100/H100/B200）

一句话总结

上游卖铲子赚现金流，中游造大脑抢入口，
下游找场景验证 PMF —
谁能把 Token 成本打下来 + 把场景做深，谁就赢下一轮。
