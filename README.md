# AI 上下游产业链 PPT

一个基于 reveal.js 的网页版演示文稿，主题为 **AI 上下游产业链全景**。

## 预览方式

直接用浏览器打开 `index.html`，或本地起一个静态服务器：

```bash
python3 -m http.server 8000
# 访问 http://localhost:8000/
```

## 内容大纲

1. 封面 — AI 上下游产业链全景
2. 议程
3. 三层结构总图（上游 / 中游 / 下游）
4. 上游 · 算力 + 芯片 + 数据 + 能源
5. 中游 · 基础大模型与平台
6. 下游 · C 端 / B 端 / 行业应用
7. 价值流动：谁在赚钱？
8. 各环节关键玩家速览
9. 关键数字
10. 四个正在发生的趋势
11. 瓶颈与风险
12. 一句话总结

## 操作快捷键

- `← / →` 翻页
- `F` 全屏
- `S` 演讲者备注
- `Esc` 幻灯片总览
